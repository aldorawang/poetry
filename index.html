Hi, there. This is an interactive website for collaborative

poetry that will make connections between people through

a creative way of making poetry together. The purpose of

this project is for people, especially poetry lovers, to

interact with each other through reading others' lines and

adding more lines into the poem. The project is based on

the human social network. Nowadays, people like to post

on social media, and there are always some fun and

creative captions. People sometimes talk in a poetic way.

There are chances for everyone to be a poet. This project

is a place for people to express and collaborate.
